### 策略说明

1. 在诸如 `香港策略`、`台湾策略` 等策略组下，可以手动选择你需要使用的自动策略组类型，如 `时延优选策略组`、`故障转移策略组` 和 `负载均衡策略组`；
2. 该配置模板内的 `地区策略` 默认首选 `时延优选策略组`；
3. 兜底策略组默认指向 `香港策略`，继承上面提及的 `香港策略` 的子策略组。
